# NoLie Leadership Hub

This GitHub Pages site contains three connected dashboards:

- `/` — Weekly Leadership Overview
- `/workshop/` — 90-Day Priority Workshop
- `/portfolio/` — Detailed Initiative Portfolio

## How the pages work together

1. Use **Workshop** to decide what earns capacity over the next 90 days.
2. Use **Portfolio** when the group needs initiative-level context, ownership, RACI, or roadmap detail.
3. Update **Home** after the workshop so the weekly operating view reflects the decisions.

## Important data note

Interactive entries are stored in the browser using local storage. They do not automatically sync across different computers or team members. Use each dashboard's Print/PDF or Export option to preserve workshop outputs.
